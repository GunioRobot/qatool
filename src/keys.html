Key Function
k show / hide key shortcuts
esc Clear current swfs, and stop timer
return Embed the currently selected swf(s)
x Embed the currently selected swf(s)
n Embed next swf
p Embed previous swf
1 Embed the first swf
9 Embed the last swf
w set background color to white
b set background color to black
c set background color to cyan
m set background color to magenta
y set background color to yellow
g set background color to green
